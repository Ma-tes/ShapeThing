# ShapeThing
![](Example0.gif)
## Description
A very simple algorithm for choosing specified symbol based on his neighbors 

## Help

* Key 'R'
    * Reshape all shapes in your buffer
 * Key 'Space'
    * Add shape to list with POINT and char. Write
